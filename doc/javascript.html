This web page is a random collection of some good JavaScript study materials. Another collection of links, many of which pertain to core JavaScript is on Rey Bango's website. Rey also has a list more targetted just to learning JavaScript.

Many commenters in the JSMentors group recommend (as of Dec. 2010) these books:

  1. JavaScript Patterns, by Stoyan Stefanov
  2. Professional JavaScript for Web Developers, by Nicholas C. Zakas

Don't forget:
JavaScript: The Good Parts

The following book is often recommended (mentioned here even though it is not really about JavaSript):
CSS: The Definitive Guide

"The top 25 books for web designers and developers"
